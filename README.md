# investment-portfolio-analyzer
Streamlit app for portfolio analysis using real market data, Monte Carlo simulation and optimization. 

## Overview

This project is a Streamlit application that analyzes investment portfolios using real market data.

It allows users to:

* Input real stock and ETF tickers
* Allocate capital across multiple asset classes
* Analyze expected return, volatility, and Sharpe ratio
* Run Monte Carlo simulations to model future outcomes
* Optimize portfolio allocation using random portfolio generation
* Visualize the Efficient Frontier (risk vs return tradeoff)

## Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* yFinance

## Features

* Real-time financial data retrieval
* Portfolio return and risk analysis
* Scenario modeling (bear, base, bull)
* Monte Carlo simulation (probability distributions)
* Portfolio optimizer (max Sharpe ratio)
* Efficient frontier visualization

## How to Run Locally

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the app:

```
streamlit run investment_portfolio_analyzer_app.py
```

## Project Purpose

This project demonstrates data analysis, financial modeling, and interactive application development for portfolio evaluation.
